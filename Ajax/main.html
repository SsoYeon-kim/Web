AJAX 함수

Web에서 화면을 갱신하지 않고 Server로부터 Data를 가져오는 방법

Browser에서 서버로 보낸 Data를 Ajax Engine을 통해 Server로 전송
이 때 Ajax Engine에서는 JavaScript를 통해 DOM을 사용하여 XMLHttpRequest(XHR)객체로 Data를 전달
이 XHR을 이용해 Server에서 비동기 방식으로 자료를 조회해 올 수 있음
Server에서 Data를 전달할 때 화면전체의 HTML을 전달하지 않고 Text 또는 XML 형식으로 Browser에 전달

Ajax의 기본 Method를 사용해(XHR 직접 사용) Server와 통신하면 복잡 -> jQuery를 이용

GET/POST 함수

$.get(URL, DATA, CALLBACK)
$.post(URL, DATA, CALLBACK)

jQuery 라이브러리에서 제공하는 '$.ajax'함수의 간편 버전
각각 GET과 POST 방식의 AJAX 요청을 보내는 데 사용
'$.ajax'함수의 설정을 미리 정의해 놓은 것으로, 더 간편하게 AJAX 요청을 보낼 수 있음

GETJSON 함수

$.getJSON(URL, DATA, CALLBACK);

JSON형식의 데이터를 불러 읽을 때 쓰이는 AJAX함수의 약식, GET 방식의 통신

GETSCRIPT 함수

$.getScript(URL);

Script파일을 불러 읽을 때 쓰이는 AJAX함수의 약식, GET 방식의 통신
